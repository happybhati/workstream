# Changelog

All notable changes to Workstream will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-27

### Added

- PR dashboard with GitHub and GitLab support (open PRs, assigned, review requests, CI status)
- PR staleness indicators (yellow 3+ days, red 7+ days)
- User avatars on PR cards
- Expandable CI check details on hover
- Jira integration with sprint tracking, task counts, yearly completion stats
- Google Calendar integration with today/tomorrow side-by-side view
- AI code review with Claude, Gemini, Ollama providers and copy-prompt option
- Review Intelligence: historical PR review collection and pattern analysis
- AI Readiness Scanner: score repos, generate bootstrapping files, create draft PRs
- Focus banner with real-time status summary
- Desktop browser notifications for key events
- Global search and keyboard shortcuts (1-5 tabs, r refresh, / search, Esc close)
- Weekly digest summary card
- Weather widget with city name
- Live animated clock
- Hourly stretch/yoga reminders
- Quick notes with retention options (forever, today, 1h, 4h, this week)
- Dark and light theme with toggle
- MCP server for AI tool integration
- CLI tool (workstream command) for service management
- macOS LaunchAgent for persistent background service
- Comprehensive configuration via .env file
